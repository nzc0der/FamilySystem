GitHub Auto-Update

Pulls the latest code from the configured GitHub remote. The database and config.json are backed up before any pull. If the pull fails, the backup is automatically restored. A successful update restarts the server via systemd.

The auto-update thread also runs every hour in the background. Manual updates are safe to trigger at any time.
